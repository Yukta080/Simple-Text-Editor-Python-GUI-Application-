# Simple-Text-Editor-Python-GUI-Application-
* Description:  
Developed a desktop-based text editor using Python’s Tkinter library, enabling users to create, open, edit, and save text files through an intuitive graphical interface.

* Key Responsibilities & Features:
- Built a responsive GUI using Python and Tkinter
- Implemented core file operations: New, Open, Save using file dialogs
- Designed user-friendly layout with menu navigation
- Handled file I/O operations with proper state updates
- Integrated message prompts for better user feedback

* Skills Demonstrated:
- GUI Development
- Event-driven programming
- File handling in Python
- Software design fundamentals
